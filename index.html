Anthropics neues KI-Modell „Mythos Preview" gilt als zu gefährlich für eine öffentliche Veröffentlichung

Claude Code Bug Fix

Anthropic gibt zu: Engineering-Fehler haben Claude Code wochenlang verschlechtert – Postmortem veröffentlicht

Nach wochenlangem Nutzer-Backlash hat Anthropic ein detailliertes Postmortem veröffentlicht: Drei separate technische Fehler verschlechterten Claude Code zwischen März und April 2026: (1) eine absichtliche, aber schlecht kommunizierte Reduktion des Reasoning-Aufwands zur Latenzkontrolle, (2) ein Session-Clearing-Bug mit Gedächtnisverlust und Wiederholungsverhalten, und (3) eine System-Prompt-Änderung gegen Verbosity, die die Coding-Qualität ruinierte. Alle drei wurden per Update v2.1.116 (20. April) behoben. Die Nutzerlimits wurden zudem für betroffene Abonnenten zurückgesetzt.

Konsequenz: Anthropic entfernte den Claude-Code-Zugang aus dem $20/Monat Pro-Plan – er ist künftig nur noch im Max-Plan enthalten.

US-Behörde GSA will eine Million Arbeitsstunden per KI automatisieren – nach 40 % Stellenabbau
